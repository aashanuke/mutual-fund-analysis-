# Bluestock Mutual Fund Analytics

## Project Overview

This project analyzes mutual fund performance, investor behavior, and investment trends using Python, Tableau, and advanced analytics techniques.

## Objectives

* Build an ETL pipeline
* Perform exploratory data analysis
* Calculate fund performance metrics
* Conduct risk analytics
* Analyze investor behavior
* Create interactive dashboards

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook
* Tableau
* GitHub

## Project Structure

data/
notebooks/
reports/
dashboard/
README.md

## How to Run

1. Install dependencies
2. Open Jupyter Notebook
3. Run ETL notebooks/scripts
4. Run analytics notebooks
5. Open Tableau workbook for dashboard
